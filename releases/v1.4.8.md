# FlowMind 1.4.8

FlowMind 1.4.8 simplifies the public entry experience so GitHub and npm readers reach the core value faster: repeatable developer workflows, reusable skills, and explicit feedback that carries forward.

## Highlights

- Rewrote the homepage README as a short entry page with clearer positioning and a faster quick start
- Split the long-form explanation into dedicated English and Chinese guide documents
- Kept the npm and GitHub entry paths aligned so readers can jump from homepage to demo, guide, or integration docs without wading through a full manual

## What Changed

- New short GitHub/npm homepage: `README.md`
- New short Chinese homepage: `README_CN.md`
- New long-form product guide: `docs/guide.md`
- New Chinese product guide: `docs/guide.zh-CN.md`

## Verification

- Re-ran core tests successfully before release: `9/9` passing
- Verified the npm package contents with `npm pack --dry-run`
- Published npm package: `flowmind@1.4.8`

## Try It

```bash
npx flowmind@latest doctor
npx flowmind@latest skills
flowmind process --skill log-audit "query the latest error logs"
```
