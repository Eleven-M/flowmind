## 🛠️ 演示故障排除

### 如果学习不生效

```bash
# 检查学习记录
flowmind learn list

# 验证配置
flowmind config --list

# 如需要重置
flowmind learn reset log-audit
```

### 如果技能无法加载

```bash
# 列出可用技能
flowmind skills

# 重新加载技能
flowmind skills --reload
```
